# VibeHist

A Python-based service for collecting and analyzing vibe coding session data through Claude Code hooks.

## Overview

Provides configurable scripts that can be integrated with `Claude Code`'s hooks system.

Capture comprehensive data about AI-assisted coding sessions, which enables analysis and observation of vibe coding behaviors.

## Features

- **Data Collection**: Capture events from `Claude Code` hooks including tool usage, session lifecycle, prompts, and responses

- **Local Storage**: Store collected data locally in files or database

- **Configurable**: Users choose which hooks to configure based on what data they want to collect
